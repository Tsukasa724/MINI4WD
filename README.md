# MINI4 SPEED KING

### アプリ概要・利用目的
アプリに登録してある店舗ごとに**ラップタイム**を投稿することが可能になっています。

さらに、ご自身が使用されているミニ四駆のマシン情報を登録していただくことができ、他ユーザーが登録したマシン情報を閲覧することもできます。

登録されている店舗情報には、営業時間・定休日・住所といった基本情報だけでなくコースレイアウトや他ユーザーのラップタイム、GoogleMAPと連携した地図情報を確認することもできるようになっています。
